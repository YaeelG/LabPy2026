DESCRIPCION GENERAL DEL SISTEMA :

"Gestión de torneo amateur
El sistema deberá permitir administrar un torneo deportivo amateur. La solución podrá contemplar registro de equipos, carga de resultados, cálculo automático de puntos, posiciones y estadísticas generales del torneo.
El sistema podrá incluir control de partidos jugados, diferencia de goles y generación de tablas de posiciones básicas."
